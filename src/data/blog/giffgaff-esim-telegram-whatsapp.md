## 🛡️ 第三步：插卡前的“保命”设置（必看！）

**请注意！在把卡插入手机之前，必须先做以下设置！** 否则插卡瞬间可能会因为系统自动验证扣掉你的余额。

### 1. 彻底切断“自动扣费”来源
iPhone 有个机制，为了保证信号，会自动用副卡的流量，或者自动发短信激活功能。我们要在**插卡前**把它们扼杀在摇篮里：

* **🚫 关闭 iMessage 和 FaceTime**
    * **路径**：`设置` -> `信息` -> 关闭 **iMessage 信息**。
    * **路径**：`设置` -> `FaceTime 通话` -> 关闭 **FaceTime 通话**。
    * **原因**：如果不关，插卡瞬间系统会自动向英国发送一条激活短信，**收费 £0.3/次**！这冤枉钱坚决不能花。

* **🚫 关闭“允许切换蜂窝数据”**
    * **路径**：`设置` -> `蜂窝网络` -> `蜂窝数据`。
    * **操作**：确保 **“允许切换蜂窝数据”** 开关是 **关闭** 的。
    * **原因**：如果不关，当你的主卡（联通/电信）信号不好时，iPhone 会自作聪明地自动切换到 Giffgaff 上网。**漫游流量 £0.2/MB**，几秒钟就能把你的余额跑光！

### 2. 插卡后的“补刀”操作
插入 Giffgaff 卡后，如果手机还是弹窗提示“运营商可能收费...”，**一定要狂点【取消】！**

## 📲 第四步：实体卡转 eSIM（iPhone用户专属）

既然是美版 iPhone，当然要发挥 eSIM 的优势，把实体卡槽留给国内主卡。

1.  **插卡**：插卡后确认有信号（中国移动/联通漫游）。
2.  **申请转换**：打开 App → `Account` → `SIM` → `Replace my SIM` → `Switch to a new eSIM`。
3.  **注意时间**：英国人的系统有作息时间，请在 **北京时间中午 12:30 以后** 操作。
4.  **写入**：按照提示点下一步，大概 2-3 分钟，旧卡失效，新 eSIM 自动写入手机。
