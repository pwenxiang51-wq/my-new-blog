**【前言】**
你想在 iPhone 上下载 YouTube 的 4K 甚至 8K 视频，但又不想买 Premium，也不想忍受满屏广告的第三方网站？

其实，你的 iPhone 比你想象的更强大。通过一个只有几十 MB 的免费 App —— **a-shell mini**，我们可以把电脑上最强悍的下载工具 `yt-dlp` 移植到手机里。

不用懂代码，跟着我这篇教程，三步就能搞定！

---

## 🛠️ 第一步：安装神器 (解决网络报错)

首先，在 App Store 搜索并下载 **`a-shell mini`** (免费，图标是个黑底白字的终端)。

打开 App，我们会看到一个类似黑客帝国的界面。别慌，我们需要先安装下载核心。因为国内网络原因，直接安装通常会报错，所以我们必须使用**清华大学的国内镜像源**来加速。

**👉 复制下方指令，粘贴到 App 里，回车：**

```bash
pip install yt-dlp -i https://pypi.tuna.tsinghua.edu.cn/simple
```

* **小白解说**：`pip` 是安装工具，`yt-dlp` 是下载软件的名字，`-i` 后面那串网址是为了让它从国内服务器下载，速度飞快且不会报错。
* **成功标志**：当屏幕最后停止滚动，并且出现 `Successfully installed...` 字样时，说明安装成功！

---

## 📥 第二步：开始下载 (注意空格！)

安装好后，以后每次下载只需要这一步。

1.  **复制链接**：去 YouTube 复制你想下载的视频链接（Share -> Copy Link）。
2.  **输入指令**：在 a-shell mini 里输入下载命令。

**⚠️ 这里的格式非常关键，90% 的人失败都是因为少敲了一个空格！**

**正确格式：**
`yt-dlp` + `空格` + `"视频链接"`

**👉 示范代码 (直接套用)：**

```bash
yt-dlp "https://youtu.be/你的视频链接xxx"
```

* **为什么加双引号？**：因为视频链接里常有 `?` `&` 等特殊符号，不加引号 App 会看不懂。
* **下载过程**：
    * 屏幕会疯狂滚动代码，显示百分比（如 `25%`... `100%`）。
    * 最后会显示 `[Merger] Merging formats`，这是它在把最高画质的画面和最高音质的音频合成一个文件。
    * 等到再次出现 `~/Documents$` 光标时，说明下载完成。

## 📺 第三步：去哪里看？用什么播放？

这是很多新手最懵的一步：“下载显示 100% 了，但相册里没有啊？视频去哪了？”

别急，iOS 的机制决定了文件在**“文件 (Files)”** App 里。

### 1. 找到视频文件
打开 iPhone 自带的 **“文件 (Files)”** App：
* 点击底部的 **“浏览”**。
* 进入 **“我的 iPhone”**。
* 找到 **“a-shell mini”** 文件夹。
* 🎉 你的视频就躺在这里！通常是 `.webm` 或 `.mp4` 格式。

### 2. 推荐播放方案 (关键！)
由于 YouTube 的高画质视频（尤其是 MKV 或 WebM 格式）不仅体积大，编码也特殊，iPhone 自带的相册有时候打不开或者没声音。

**强烈推荐配合以下软件观看：**

* **方案 A：VLC for Mobile (免费、全能)** ⭐⭐⭐⭐⭐
    * 去 App Store 下载 **VLC**。
    * 在“文件”App 里长按视频 -> **共享** -> 选 **VLC**。
    * **优点**：通吃所有格式，支持 4K/8K HDR，支持外挂字幕，完全免费。

* **方案 B：nPlayer (付费但强大)** ⭐⭐⭐⭐⭐
    * 如果你是老牌 iOS 用户，肯定知道它。
    * 直接在 nPlayer 里连接本地文件播放，解码能力比 VLC 更强，支持杜比音效。

* **方案 C：存入系统相册 (仅限 MP4)**
    * 如果你下载运气好是 `.mp4` 格式：
    * 在“文件”里打开视频 -> 左下角 **分享按钮** -> **“保存视频”**。
    * 这样它就会出现在你的照片 App 里了。

## ❓ 常见问题急救

**Q：提示 `command not found`？**
A：检查一下指令中间是不是**漏了空格**！比如 `yt-dlp"https..."` 是错的，必须是 `yt-dlp "https..."`。

**Q：提示 `SSL module is not available`？**
A：这是网络问题。尝试切换一下你的 WiFi 或者流量，通常就能解决。

**Q：能下载 B站 吗？**
A：**当然可以！** `yt-dlp` 支持全球几百个视频网站。把链接换成 B站 的，操作一模一样。
