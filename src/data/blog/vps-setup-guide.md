### 🌐 1️⃣ 第一步：域名解析 (Cloudflare)

如果你打算使用 **Hysteria2** 或 **TUIC v5** 这类需要真实证书的高速协议，我们需要先给 VPS 分配一个“门牌号”。（注：如果你只用 Reality 协议，此步可跳过，但建议配置以解锁全部姿势）。

1. 登录 Cloudflare 后台，点击你的域名。
2. 进入 **DNS** -> **记录 (Records)**。
3. ➕ **添加记录 (Add record)**，按下图填写：

| 类型 | 名称 | 内容 | 代理状态 |
| :--- | :--- | :--- | :--- |
| A | vps | 你的VPS IP | **仅 DNS (灰色云)** |

> **⚠️ 重点提醒**：**一定要把“小黄云”点灰**！否则后续脚本无法为你自动申请 SSL 证书。

### 💻 2️⃣ 第二步：使用 CMD 连接 VPS

Windows 用户不需要下载第三方软件，直接用系统自带的 CMD 即可。Mac 用户直接打开“终端”。

1. 按 `Win + R`，输入 `cmd` 回车。⌨️
2. 输入连接命令（将下面信息替换为你自己的 IP 和端口）：

```bash
# 格式：ssh root@你的IP -p端口号 (默认端口通常是22，可省略 -p22)
ssh root@192.168.1.1
```

3. 输入密码登录🔑（提示：密码输入时屏幕上不会显示任何字符，这是正常的，盲输完直接回车即可）。
