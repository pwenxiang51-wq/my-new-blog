## 💡 这不是探针，这是武装到牙齿的“赛博装甲”

Velox 绝不是一个单纯的监控脚本，我把它定位为**「全域运维与容灾备份」的终端级 OS**。我将整整 **27 项** 杀手级极客操作，强行压缩进了一个极致清爽的双轨制 UI 面板中。

小白用它当全能保姆，老鸟用它当逆天神器。下面随便掏出几个核心功能，懂行的自然懂：

### 🛡️ 杀手锏一：抓外鬼与 SSH 隐身防盗门
你的机器每天都在被自动化脚本疯狂扫端口？Velox 内置了工业级的系统安全审计中心：
* 🕵️ **底层抓鬼**：一键穿透系统底层调取日志，精准揪出 Top 10 恶意爆破攻击 IP。
* 💣 **物理制裁**：一键透视当前所有在线的 SSH 用户，发现异常连接，直接强制踢出并**写入系统防火墙永久拉黑**（甚至可以给对方的终端发送红色“死神警告”恶搞弹窗）。
* 🚨 **神盾预警**：一键无痛部署 Telegram 机器人报警。内置独创的**「物理主网卡强制绕过技术」**，无论你的机器套没套 WARP，只要有人登录或机器开机，你的手机会瞬间收到包含 4 大代理核心状态的精美体检报告！

### 🧳 杀手锏二：全域资产打包，跨机“秒级”克隆搬家
买新机器了？配置节点、搞面板、续签证书能折腾你一整晚！在 Velox 这里，只需要 1 分钟。
* 📦 **高阶提取**：底层强植入 `cp --parents` 逻辑，一键暴力扫描并打包你系统内的节点配置、X-UI 数据库、Acme 证书，甚至自定义网盘目录（绝对路径 1:1 完美锁定，毫不差池）。
* ⚡ **极速克隆**：自动为你生成一段专属于你的**跨机克隆长指令**。登录新机器，粘贴回车，所有数据、面板、证书瞬间覆盖，秒级满血复活，直接起飞！

### 🌐 杀手锏三：代理维稳与底层高阶网络排雷
* 🎯 **IP 纯净度双轨雷达**：别再被机场主忽悠了！一键透视你的出口 IP 到底是“极品原生家宽”、“普通机房”还是“被拉黑的高欺诈风险”。
* 🔗 **节点提权雷达**：全盘全局正则扫描机器内隐藏的 VLESS / VMess / Hy2 / AnyTLS 节点，一键生成 Base64 聚合订阅和终端专属迷你二维码，手机打开直接扫。
* 🚀 **TCP 暴力扩容**：嫌网速慢？一键强行突破系统限制，暴力扩展底层 TCP 读写窗口缓冲区，让大文件下载起步速度直线狂飙；更支持一键注入 BBR 加速、Argo 隧道与 WARP 出站状态的智能探活与假死修复。

## 🚀 极速部署 (One-Line Install)

真男人从不磨叽，直接上终端！
请在你的 VPS 命令行中（推荐使用 `root` 权限），复制并执行以下一键安装指令：

```bash
bash <(curl -sL https://raw.githubusercontent.com/pwenxiang51-wq/Velox-VPS-Panel/main/install.sh)
```

**💡 极客使用说明：**
部署一旦成功，这台机器就已被你彻底接管。以后无论何时登录机器，只需在终端优雅地敲下 `velox` 并按回车，即可瞬间唤醒这套航母级面板！
