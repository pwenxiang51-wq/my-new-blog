很多人的电脑用久了，C 盘就会变成红色的“存储预警”。最让人抓狂的是，明明把软件都装在 D 盘了，C 盘还是会莫名其妙地变满。

其实，真正的“空间杀手”并不在你的桌面上，而是一个隐藏文件夹：`AppData`。

今天教大家如何安全地清理这些“隐形垃圾”，瞬间给 C 盘回血。

### 1. 找到“隐形杀手”

首先，我们需要进入这个隐藏目录。
按下键盘上的 `Win + R` 键，输入 `%appdata%`，然后回车。你会直接进入 `Roaming` 文件夹，往上一级就是 `AppData`。

这里面通常有三个文件夹，最占空间的是 `Local` 和 `Roaming`。

### 2. 第一号嫌疑人：微信/QQ (WeChat/Tencent)

如果你在电脑上用微信，它绝对是 C 盘第一大户。它默认会把所有的聊天记录、图片、视频都存在 C 盘。

* **位置**：`Documents\WeChat Files` 或 `AppData\Roaming\Tencent`
* **解决方法**：
    1.  打开电脑版微信设置。
    2.  找到“文件管理”。
    3.  点击“更改”，把路径迁移到 D 盘。
    *（这一步操作完，通常能腾出 5G-20G 不等的空间！）*

### 3. 第二号嫌疑人：浏览器缓存 (Chrome/Edge)

浏览器为了让你下次打开网页快一点，会把你看过的图片、视频缓存下来。久而久之，这就是个巨大的垃圾堆。

* **位置**：`AppData\Local\Google\Chrome\User Data`
* **解决方法**：
    * **笨办法**：直接在浏览器里按 `Ctrl + Shift + Delete`，选择“清除缓存图片和文件”。
    * **彻底办法**：卸载浏览器，重装时选择安装到其他盘（虽然很多浏览器强制装 C 盘，建议定期清理缓存）。

### 4. 第三号嫌疑人：系统临时文件 (Temp)

这是 Windows 运行各种软件时产生的“草稿纸”，用完往往不记得扔。

* **位置**：`AppData\Local\Temp`
* **操作**：
    1.  全选这个文件夹里的所有文件。
    2.  直接 `Shift + Delete` 永久删除。
    3.  如果提示“文件正在使用”，勾选“跳过”即可。

### 5. 终极神器：WizTree (强烈推荐)

如果你懒得一个个找，去下载一个叫 **WizTree** 的免费小软件。

它比系统自带的清理工具快 100 倍。它会用一个个“色块”展示你的硬盘（如下图示意），色块越大，代表文件越大。一眼就能看出是谁在占着茅坑不拉屎，右键直接删除即可。

### ⚠️ 避坑指南

在清理 `AppData` 时，切记：
* **不懂的文件夹不要乱删**：特别是 `Local\Microsoft` 下的文件，删错了可能导致系统蓝屏。
* **先迁移，后删除**：像微信、Adobe 这种软件，尽量在软件设置里改路径，而不是暴力删除文件夹。

给 C 盘留出至少 20GB 的余量，你的电脑运行速度会肉眼可见地变快！
